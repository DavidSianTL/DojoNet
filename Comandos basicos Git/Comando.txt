Comando	Descripción
git config --global user.name "Tu Nombre"	Configura tu nombre de usuario en Git.
git config --global user.email "tuemail@example.com"	Configura tu correo en Git.
git config --global credential.helper store	Guarda tus credenciales para no ingresarlas en cada push.
git config --list	Muestra la configuración global de Git.

Inicialización y Clonación
Comando	Descripción
git init	Crea un nuevo repositorio Git en la carpeta actual.
git clone URL_DEL_REPOSITORIO	Clona un repositorio remoto en tu máquina local.

Estados y Seguimiento de Archivos
Comando	Descripción
git status	Muestra el estado de los archivos (modificados, nuevos, en seguimiento).
git add archivo.txt	Agrega un archivo específico al área de preparación (staging).
git add .	Agrega todos los archivos modificados al staging.
git reset archivo.txt	Saca un archivo del área de preparación.
git rm --cached archivo.txt	Deja de rastrear un archivo en Git sin eliminarlo.

Commits y Registro de Cambios
Comando	Descripción
git commit -m "Mensaje"	Crea un commit con los cambios añadidos al staging.
git commit --amend -m "Nuevo mensaje"	Modifica el último commit (si olvidaste algo).
git log	Muestra el historial de commits.
git log --oneline	Muestra el historial de commits en una línea.

Ramas (Branches)
Comando	Descripción
git branch	Lista las ramas locales.
git branch -a	Lista todas las ramas (locales y remotas).
git checkout nombre-rama	Cambia a otra rama.
git switch nombre-rama	Nueva forma de cambiar de rama en Git.
git checkout -b nueva-rama	Crea una nueva rama y cambia a ella.
git branch -d nombre-rama	Elimina una rama local.

Sincronización con Repositorios Remotos
Comando	Descripción
git remote -v	Muestra los repositorios remotos vinculados.
git remote add origin URL	Agrega un repositorio remoto.
git fetch origin	Descarga cambios del remoto sin fusionarlos.
git pull origin rama	Descarga y fusiona los cambios de una rama remota.
git push origin rama	Envía cambios de tu rama local al repositorio remoto.
git push -u origin rama	Sube cambios y establece el seguimiento con el remoto.

 Fusión y Rebase de Ramas
Comando	Descripción
git merge nombre-rama	Fusiona una rama con la actual.
git rebase nombre-rama	Aplica los cambios de otra rama sobre la actual.

 Revertir Cambios
Comando	Descripción
git checkout -- archivo.txt	Revierte cambios en un archivo (antes del add).
git reset HEAD archivo.txt	Deshace git add en un archivo.
git reset --hard HEAD~1	Deshace el último commit sin posibilidad de recuperarlo.
git revert HEAD	Crea un commit inverso para deshacer el último cambio.

Tags (Etiquetas de Versionado)
Comando	Descripción
git tag	Lista todas las etiquetas (tags).
git tag -a v1.0 -m "Versión 1.0"	Crea una etiqueta con mensaje.
git push origin --tags	Sube todas las etiquetas al repositorio remoto.
